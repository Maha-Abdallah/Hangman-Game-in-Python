# Hangman-Game-in-Python

# Description
This repository contains a simple Hangman game implemented in Python. The game randomly selects a word from a predefined list, and the player guesses letters to reveal the word. Itâ€™s a great starting point for beginners who want to practice Python programming and game development.

# Features
Random word selection
Displaying the word with underscores for unguessed letters
User input for guessing letters
Winning condition when all letters are guessed

# Installation
Clone this repository.
Run the hangman.py script in your terminal or IDE.

# Usage
Execute the script.
Guess letters by typing them in the terminal.
Keep guessing until you reveal the entire word.
